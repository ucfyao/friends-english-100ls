# Anki 使用指南 — Friends 英语学习

## 1. 下载安装

| 平台 | 地址 | 费用 |
|------|------|------|
| Mac / Windows / Linux | https://apps.ankiweb.net | 免费 |
| Android | Google Play 搜 AnkiDroid | 免费 |
| iOS | App Store 搜 AnkiMobile | ¥168 |

注册 AnkiWeb 账号可以多设备同步。

## 2. 导入 Friends 词汇卡

1. 打开 Anki → **File → Import**
2. 选择文件：`英语学习/Friends-Anki.tsv`
3. 导入设置：
   - Type: **Basic**
   - Deck: 新建 `Friends English`
   - Fields separated by: **Tab**
   - 字段映射：

| 文件列 | Anki 字段 | 说明 |
|--------|----------|------|
| Front | Front | 单词 |
| Phonetic | 新建字段 | 音标 |
| 中文释义 | 新建字段 | 中文意思 |
| English | 新建字段 | 英文释义 |
| 总次数 | 新建字段 | 全剧出现次数 |
| 出现集数 | 新建字段 | 跨多少集 |
| Contexts | Back | 剧中原句（显示在背面）|
| 出现详情 | 新建字段 | 每集出现几次 |

4. 点 **Import** 完成

## 3. 开启 FSRS 算法

1. 点击 deck 旁边的齿轮 → **Options**
2. 往下滑找到 **FSRS** 开关，打开
3. 设置 **Desired retention**: `0.9`（推荐值，记住率 90%）
4. 点 **Save**

## 4. 每日学习流程

```
刷 Anki 闪卡（10-15min）→ 看对应集 Friends（开英文字幕）→ 循环
```

### 刷卡要点

- 看到正面（单词），先在脑中回忆意思
- 翻转看背面，只需按两个键：
  - **Again** — 完全不记得
  - **Good** — 想起来了
- 不用纠结 Hard / Easy，两个按钮足够
- FSRS 会自动计算最佳复习间隔

### 推荐节奏

| 阶段 | 新卡/天 | 配合 |
|------|--------|------|
| 第 1 周 | 20 张 | 看 S01E01-E03 |
| 第 2-4 周 | 30 张 | S01 剩余集数 |
| 之后 | 40-50 张 | 按季推进 |

## 5. 先刷卡再看剧

顺序很重要：**先刷卡 → 再看剧**

- 先把关键词装进短期记忆
- 看剧时在真实语境中命中
- 大脑判定"重要"，写入长期记忆

反过来效果差很多。

## 8. Obsidian 台词阅读技巧

每集台词中的生词已经标注了链接，可以快速查看释义：

| 词汇分级 | 台词中的显示 | 说明 |
|---------|------------|------|
| 出现 >=50 集 | 无标注 | 太常见，不打扰阅读 |
| 出现 10-49 集 | 带链接（可点击） | 点击跳转到总表查释义 |
| 出现 <10 集 | 带链接 + 音标 | 直接看到发音，点击查释义 |

### 悬停预览（推荐）

不需要跳转就能看到释义：

1. **按住 `Cmd` 键**（Mac）或 `Ctrl` 键（Windows）
2. **鼠标悬停**在台词中带链接的单词上
3. 会弹出浮窗，直接显示总表中的释义卡片（音标、中英释义、全剧出现次数）

这样读台词时遇到生词，按住 `Cmd` 晃一下鼠标就能看懂，不用离开当前页面。

> 此功能由 Obsidian 内置的 **Page Preview** 核心插件提供，默认已开启。
> 如未生效，检查：Settings → Core Plugins → Page Preview 是否打开。

## 9. 常见问题

**Q: 一天刷不完怎么办？**
减少每天新卡数量，但复习卡一定要清完。复习 > 新学。

**Q: 近 8000 张卡要刷多久？**
每天 30 张新卡，约 9-10 个月刷完全部。配合 FSRS，实际每天复习量在 100-150 张左右，大约 15 分钟。

**Q: 手机和电脑怎么同步？**
注册 AnkiWeb 账号，电脑端点 Sync，手机端登录同一账号自动同步。
